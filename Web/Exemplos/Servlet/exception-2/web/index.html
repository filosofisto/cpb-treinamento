sendError

setError é usado para indicar situação que de ERRO

Envia um erro ao client usando o código de erro especificado e limpa o buffer. O servidor preservará os cookies e limpará ou atualizará qualquer header necessário a página de erro. Se uma página de erro estiver configurada ao código de erro especificado o servidor retornará a página de erro.

SC_NOT_FOUND
SC_UNAUTHORIZED
SC_INTERNAL_SERVER_ERROR
